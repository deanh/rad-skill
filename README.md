# Radicle Skill for Claude Code

A Claude Code plugin for working with [Radicle](https://radicle.xyz) - a peer-to-peer code collaboration protocol.

## Installation

Add to your project's `.claude/settings.json`:

```json
{
  "plugins": ["git@github.com:deanh/rad-skill.git"]
}
```

Or install globally in `~/.claude/settings.json` to make it available in all projects.
